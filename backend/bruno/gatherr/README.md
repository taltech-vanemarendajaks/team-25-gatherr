# Gatherr - Bruno Collection

Gatherr backend API requests 

### Prerequisites

[Bruno API client](https://www.usebruno.com/)

### Setup

Open this folder as a collection and select the **local** environment.

Get `token` from Google Login (any non-empty string works as `idToken` locally) and paste it into the `token` secret in the environment editor so it can be used by other requests.
